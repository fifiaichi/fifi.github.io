Python Claude

AI辅助批量工具开发

作品集网站包含 6 个项目、40+ 个中文命名的资源文件。手动重命名和修改 HTML 引用极易出错。我用 Claude 辅助编写 Python 脚本，实现一键批量重命名 + 全站路径自动替换，支持预览模式确保安全。

# 批量重命名 + 路径替换
RENAME_MAP = {
"image/animation/home-video.mp4":
"image/animation/home-video.mp4",
...
}
shutil.move(old, new) # 执行重命名
content.replace(old_path, new_path)
安全预览模式 → 确认后执行 → 自动生成 .bak 备份
